# Психологический портрет Руса

## 1. Фон (детство Поток, resilience)
Родился/вырос в Барнауле, район "Поток" (1991): гоп-стопы, проблемные подростки, терроризирующие школу. Жёсткая среда → resilience, выживание. Семья: отец — жёсткий, злость на "глупые вопросы/ошибки" (триггер на нерациональность); мама — саботаж амбиций ("зачем Москва/тренерство"), бессмысленные разговоры → избегание общения, ощущение "разных миров".

## 2. Профессия/интересы (DDX, TS, Insta, доход 100k+)
Руководитель фитнес-клуба/тренер (DDX Fitness), TypeScript-разработчик, видео/нейроконтент, развитие Instagram. Фокус: спорт/ПП, бизнес/инвестиции, пассивный доход >100k ₽/мес, бизнес-модели (лиды косметика, онлайн-школы). Капитал ~15 млн руб.

## 3. Сильные стороны (лидер, честность, рефлексия)
Лидер (помог коллегам, ушёл по соглашению), честность/рациональность (факты, критика рисков), рефлексия (терапия паттернов, аффирмации), resilience (из Потока), проактивность (оптимизация, автоматизация).

## 4. Триггеры/паттерны (отвержение, дисбаланс отношений, редкие глубокие разговоры)
- Отвержение/обесценивание вклада (отец/мама → проекция на девушку/сотрудников).
- Дисбаланс отношений: ревность (страх abandonment у девушки-VDA, злость за вечер с коллегой); ретрите Турция — несоответствие ожиданий (молчание/пассивная агрессия вместо диалога).
- Редкие глубокие разговоры: мало понимают трансформацию/жизнь (пример: DDX-коллега после комы/терапии, прощает предательство жены).
- Деньги=свобода: крупные траты (50k+ подарки) → стресс "банкомат?".

## 5. Рекомендации (баланс, рост)
- Границы: "Ошибки других — норма", "проси прямо", языки любви (Чепмен).
- Терапия: EMDR (отец/мама), когнитивка (деньги=свобода); для пары — группы VDA, timely фидбек.
- Рост: networking с "глубокими" (DDX-коллега как терапевт/партнёр), баланс (работа/отношения/само), масштабировать пассивку >100k.
