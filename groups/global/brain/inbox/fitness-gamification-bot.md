# Telegram-бот геймификации для фитнес-клубов (White-label SaaS)

## 🎯 Проблема
- 80% клиентов фитнес-клубов бросают тренировки через 2 месяца (лень > мотивация).
- Клубы теряют 70-80% дохода от churn'а ежегодно.
- Средний чек 3-5к руб/мес на клиента → удержание +10% = миллионы выручки.

## 💡 Решение
**White-label Telegram-бот для клубов** (продаём B2B, не конечным юзерам):
- **QR-чек-ин** на ресепшен (динамический QR от бота, уникальный на день/сессию).
- **Виртуальный персонаж** (Tamagotchi для качков): 

**Персонализация (2026-04-07, Рус):**
- User кидает фото → Midjourney шарж (prompt: "cartoon gym newbie caricature from photo traits, RPG style --ar 1:1 --sref <fixed-style>").
- Хранение: шарж PNG (не фото, GDPR-safe).
- Check-in: remix +5% к buff-hero ref (плавно 0-100%).
- Стиль lock: один --sref для consistency всех аватаров.
  - Выбор: мужчина/женщина/унисекс.
  - Прокачка: +10 баллов/визит, +50 за стрик 3+ дня, уровни, бейджи.
- **Геймификация**: ежедневные квесты ("посети 3 раза на неделе"), лидерборд зала.
- **Админ-панель для владельца**: дашборд посещаемости, ROI-метрики (% роста retention).

**MVP фичи** (не перегружать):
- Регистрация юзера → выбор перса.
- /checkin → QR-скан → прокачка.
- /profile → статус перса.
- /stats → общая статистика.

## 💰 Монетизация
- **Freemium**: первые 100 юзеров бесплатно.
- **Подписка**: 10к руб/мес базовый, 20к премиум (кастом персы, аналитика).
- **Upsell**: интеграция с их CRM, веб-версия.
- **Потенциал**: 10 клубов = 100к+/мес чистыми. 50 клубов = 1млн/мес пассивка.

## 📊 Рынок
- РФ: 10к+ фитнес-клубов (Москва/СПб ~50%).
- ЦА: мелкие/средние залы (1-5 локаций) без своих devs/аппов.
- Конкуренты: WorldClass имеют аппы → игнорим гигантов, фокус на independents.
