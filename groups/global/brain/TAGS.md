# Каталог тегов Second Brain

> AI выбирает теги только из этого списка. Добавить новый тег = осознанное решение, не автоматика.

Тот же набор зашит в `workspace/skills/brain-capture/scripts/capture.js` (`TAG_CATALOG`) — при расширении каталога обновляй оба места.

## Домены (о чём заметка)

| Тег          | Когда использовать                                 |
| ------------ | -------------------------------------------------- |
| `бизнес`     | Предпринимательство, модели монетизации, стратегия |
| `маркетинг`  | Продвижение, аудитория, контент, воронки           |
| `продукт`    | Фичи, UX, разработка продукта                      |
| `финансы`    | Деньги, расходы, инвестиции, бюджет                |
| `технологии` | AI, инструменты, стек, автоматизация               |
| `люди`       | Команда, партнёры, клиенты, нетворк                |
| `обучение`   | Знания, курсы, книги, навыки                       |
| `личное`     | Личный опыт, привычки, состояние                   |
| `проект`     | Конкретный текущий проект                          |

## Тип инсайта (какой вывод)

| Тег           | Когда использовать              |
| ------------- | ------------------------------- |
| `паттерн`     | Повторяющаяся закономерность    |
| `возможность` | Незакрытая ниша, точка роста    |
| `риск`        | Угроза, красный флаг            |
| `гипотеза`    | Идея которую нужно проверить    |
| `ошибка`      | Что пошло не так и почему       |
| `принцип`     | Правило которое работает всегда |

## Горизонт (когда актуально)

| Тег         | Когда использовать                       |
| ----------- | ---------------------------------------- |
| `сейчас`    | Актуально прямо сейчас, требует действия |
| `позже`     | Интересно, но не срочно                  |
| `архив`     | Контекст прошлого, уже неактуально       |
| `evergreen` | Не устаревает, фундаментальное знание    |
